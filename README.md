# ❤️ Cardiovascular Risk Prediction

A machine learning web application that predicts whether a person is at
High Risk or Low Risk of cardiovascular disease.

## 🔗 Live Demo
👉 [Click here to try the app](your-render-link-here)

---

## 📌 About
This is a Final Year BCA Project that uses Machine Learning to predict
cardiovascular disease risk based on basic health and lifestyle information
like age, BMI, smoking habits, diabetes, and more.

---

## 🛠️ Built With
- Python
- Flask
- XGBoost
- Scikit-learn
- HTML & CSS

---

## 📊 Dataset
- Source: Kaggle
- Size: 300,000+ patient records
- Features: 19 health and lifestyle attributes

---

## ⚙️ Run Locally
1. Clone the repo
```
git clone https://github.com/yourusername/Cardiovascular-Risk-Prediction.git
```

2. Install dependencies
```
pip install -r requirements.txt
```

3. Run the app
```
python app.py
```

4. Open browser and go to
```
http://localhost:5000
```
